    (2) What are the specific volcano attributes that cause a volcano to be more susceptible to tidal forces?

    Dominant Rock Type and Tectonic Setting

    The results varied for different phases, but as a whole, the features that were most important were the dominant rock type and the tectonic setting.     (3) How well can we predict the phase at which a volcano will erupt based on volcanic and eruptive features?

    Not extremely well, but better than no prediction power.

    Considering the scores we received on the Random Forest test, it is very hard to predict volcanic eruptions when we examine the data from the standpoint that we used. We probably would have been better off using some form of Baye's Rule, however given the sparsity of eruptions for many volcanoes, we would not have the proper amount of data that we would need to build a really good classification model. Hence, if we used Bayes' Rule, we would do something to the tune of:

    What is the probability of an eruption occurring given that the volcano has feature X,Y,Z, is within this phase of the gravitational force and erupts according to this frequency.

    The highest score was a .57, which does not given us a very good F1 score.

    (4) Which tidal force correlates the strongest with volcanic eruptions?

    Solar Semi-annual

    The semi-annual phase correlates the strongest, which seems to suggest that volcanoes eruption rates correlate strongest to the semi-annual period. This is consistent with visual histograms found in the studies that we first encountered when we embarked on this journey.
